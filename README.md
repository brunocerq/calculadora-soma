Esta app (app.py) resolve o problema de encontrar um subconjunto de elementos da Coluna A e um subconjunto de elementos da Coluna B tal que a soma de ambos os subconjuntos seja exatamente igual e ao mesmo tempo corresponda à maior soma possível entre os elementos dos dois conjuntos. (Programação Linear Inteira) O número de elementos usados de A e B podem ser diferentes. A app recebe um excel com conjunto A/B e devolve em CSV os elementos usados nas somas de cada conjunto. 
A versão (app-mais-info) recebe dois conjuntos A/B e processa ambos, para além de retornar a maior soma possível, devolve em CSV os elementos NÃO usados nas somas de cada conjunto, para além de alguma info adicional de cada um.
